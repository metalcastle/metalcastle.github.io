hexo new创建文章后用指定编辑器打开

发表于 2019-07-12

测试系统：win10

hexo版本：3.8.0

在hexo目录下找到scripts目录，没有就新建一个。

打开scripts目录，新建一个文本文档，重命名为”Hexo添加文章时自动打开编辑器.js”，然后用记事本打开这个文件，复制和粘贴以下代码。

var spawn = require(‘child_process’).exec;

// Hexo 3 用户复制这段
hexo.on(‘new’, function(data){
spawn(‘start “C:\Program Files\Notepad++\notepad++.exe” ‘ + data.path);

});

注意你要使用的编辑器的路径要用双斜线，否则会没有反应或者打开后是个命令行窗口。我使用的是Notepad++，这个打开速度会比较快一些。
